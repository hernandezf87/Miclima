# MiClima

MiClima is a weather application that displays real-time weather data alongside
daily positive affirmations, enhanced with background video and smooth animations.

## Features
- Real-time weather data using OpenWeather API
- Background weather videos
- Daily affirmations
- Persistent local storage

## Architecture
- MVC architecture
- Network layer using URLSession and Codable
- Local persistence using Core Data

## Tech Stack
- Swift
- UIKit - (curretnly updating to SwiftUI)
- URLSession
- AVKit / AVFoundation
- Core Data

## Technical Highlights
- Asynchronous network requests with proper error handling
- Media playback optimized for performance
- Data persistence for offline usage
